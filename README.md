# 🎬 Снимок кадра + Текст (Video Frame Capturer)

Легкое одностраничное веб-приложение для захвата кадров из видео и добавления настраиваемого текста. 

## 🛡️ Приватность и Офлайн-режим
Приложение работает **полностью в вашем браузере**. Ваши видеофайлы и кадры **никогда не загружаются** на какие-либо серверы. Вы можете использовать его без подключения к интернету после первой загрузки страницы.

## ✨ Возможности
- 📂 Загрузка видео перетаскиванием (Drag & Drop) или через диалог выбора файла.
- ⏮️ ⏭️ Покадровая прокрутка для точного выбора момента.
- 🖼️ Мгновенный захват кадра в исходном разрешении видео.
- ✏️ Добавление текста с возможностью **перетаскивания** мышкой или пальцем (на мобильных).
- 🎨 Настройка шрифта, размера, цвета и черной обводки для читаемости.
- 💾 Экспорт результата в форматы **PNG, JPEG или WebP** с настройкой качества.
- 📱 Адаптивный дизайн, работает на ПК и мобильных устройствах.

## 🚀 Как использовать
1. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).
2. Перетащите видеофайл в рабочую область или нажмите кнопку "Открыть видео".
3. Используйте кнопки `−1 кадр` / `+1 кадр` для точной настройки момента.
4. Нажмите `📸 Сделать снимок`.
5. Введите нужный текст, настройте его стиль и перетащите в нужное место на кадре.
6. Выберите формат и нажмите `💾 Скачать кадр`.

## 🛠️ Технологии
- Чистый HTML5, CSS3 и Vanilla JavaScript (без внешних зависимостей и фреймворков).
- Canvas API для рендеринга и экспорта изображений.
- File API и Blob URL для локальной работы с файлами.
